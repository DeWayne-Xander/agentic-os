Most Agent OS setups run on one laptop. That works if you sit at the same desk every day. It does not work if you have three offices. It does not work if you travel. It does not work if you switch between a desktop and a laptop and a phone.

VI. How John did it

The 8-step setup. Plain English.

You don't need to be technical. You need to be patient for 90 minutes. Most members get through this on a Sunday morning with a coffee.

1.

Get a Hostinger VPS

Pick the smallest plan. Linux Ubuntu. About $5 a month. Hostinger gives you root access, a public IP, and a 1-click web terminal. That's all you need.

2.

Install Node + the basics

One command — apt install nodejs git curl. You're getting the same Node 20 the laptop install uses. Plus a few system tools.

3.

Upload the Agent OS source

Open the agent-os-pack.zip. Drag the source/ folder onto the VPS via SFTP. No GitHub, no clone — just the files you already have.

4.

Run npm install then npm run dev

Same two commands as on a laptop. The dashboard now runs on the VPS at port 3737. You can't see it yet — Cloudflare fixes that next.

5.

Install cloudflared + create a tunnel

One install. One cloudflared tunnel create hermes. Cloudflare gives you a token. You drop it into a config file. The pipe is open.

6.

Point a subdomain at the tunnel

In Cloudflare DNS, add a CNAME like dash.yourdomain.com pointing at the tunnel. Cloudflare handles HTTPS for free.

7.

Turn on Cloudflare Access

In Cloudflare Zero Trust, create an Access policy for dash.yourdomain.com. Allow only your email. Cloudflare sends a one-time code on every login. The dashboard now requires it.

8.

Run it as a service so it never sleeps

Use pm2 or a tiny systemd unit. The dashboard now starts on boot, restarts on crash, and runs forever. You close every laptop. It keeps going.

Q6.
Sachi Lord — YouTube thumbnail skill?
read on Skool →
"Does anyone have a YouTube Thumbnail creation skill they're willing to share? Something that looks at competitor thumbnails on similar topics, sees which videos are doing well, creates 20 variations of it."
My answer —

Yes. This is a Hermes + Studio combo skill. I'll drop it in the Boardroom this week, but the shape is this:

Step 1 — Scrape. Hermes uses the YouTube Data API. Pass in a topic. It pulls the top 50 videos by view count from the last 90 days. Gets every thumbnail URL, view count, channel, age, title.

Step 2 — Score. Hermes ranks them by views-per-day, not just total views. New videos with high velocity beat old videos with stale view counts.

Step 3 — Style-extract. Pass the top 5 thumbnails to gpt-image-2. Ask it to describe the visual style — colours, composition, expression, text placement.

Step 4 — Generate. Studio (the Hermes image agent) creates 20 variations of YOUR thumbnail in the extracted style. Same face, same headline — 20 different compositions.

Then you pick one. The whole loop runs inside Agent OS in about 4 minutes.
